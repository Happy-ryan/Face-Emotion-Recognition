# Face-Emotion-Recognition

## **💡프로젝트 요약**
인간의 감정을 구분하는 딥러닝 모델을 AU스코어를 추출하고 다중 로지스틱 회귀를 이용하여 설명함으써 분류의 근거를 얻는다.

## **💡사용법**
1. 폴더 생성하기
```
creatFolder 코드 깃허브 주소
```
2. 도커 다운로드 및 실행 
```
https://docs.docker.com/get-docker/
```
3. 실행
```
exec 코드 깃허브 주소
```
- 실행결과
! exec 파일 ipynb 또는 이미지 파일
